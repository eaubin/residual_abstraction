# Residual Abstraction

This repository studies how a transformer residual stream relates to the
distribution of completions it induces.

## Where to Start

- `EXPERIMENTS.md` is the experiment index and current status.
- `FORMALISM.md` defines the named quantities, verdict predicates, and scope
  constraints used by the experiments.
- `experiments/` contains the canonical per-experiment registrations and
  conclusions.
- `AGENTS.md` records the working norms and research commitments for future
  agents.
